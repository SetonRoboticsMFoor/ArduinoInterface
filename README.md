# ArduinoInterface

The index.html should be hosted using Github pages or another hosting method to create the static web page. 

The Arduino code should be pasted into the Arduino sketch and uploaded to the Arduino.

The web interface allows you to connect the computer to the Arduino via the normal USB connection. Be sure to disconnect the serial monitor or the web interface will not work. You can use the web interface to turn on or off the LED and change the brightness. The brightness uses gamma correction to make the brightness appear linear on the slider.
